# Faith Business Dashboard

Your automated Christian business command center — built for phone, tablet, and laptop.

## Stack
- **Next.js 14** (App Router)
- **Supabase** — faith-business project
- **Tailwind CSS** — dark gold aesthetic
- **Recharts** — revenue charts
- **PWA** — installable on any device

## Pages
- `/` — Dashboard (revenue, profit, chart, seasonal alerts)
- `/approval` — Review queue (approve/reject product ideas, copy, images)
- `/products` — Full catalog with search and filters
- `/trends` — Seasonal calendar + market opportunities
- `/tax` — Quarterly tax tracker with set-aside calculator

## Deploy to Vercel
1. Import this repo in Vercel
2. Add env vars: NEXT_PUBLIC_SUPABASE_URL and NEXT_PUBLIC_SUPABASE_ANON_KEY
3. Deploy

## Install as PWA
- iPhone: Safari → Share → Add to Home Screen
- iPad: Safari → Share → Add to Home Screen
